# Particle Garden

A single-page interactive particle playground rendered on an HTML canvas. Click and drag to spawn and steer particles across five different behavior modes.

## Running it

No build step or dependencies — just open `index.html` in a browser.

## Controls

- **Modes**: Attract, Repel, Flock, Fountain, Fireworks — changes how particles react to your pointer.
- **Palette**: choose between Rainbow, Sunset, Ocean, and Neon color swatches.
- **Size** / **Trail** sliders: adjust particle size and how long trails linger.
- **Clear** button: wipes the canvas and all active particles.

Works with both mouse and touch input.
